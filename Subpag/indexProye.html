Sistema para caracterización de señales ECG

-- Tesis -- 2019 --

Este proyecto nacío a raíz de la necesidad de una aplicación de telemedicina para la detección de anomalías cardíacas dentro de él examen ECG (Electrocardiograma), su función es adquirir señales ECG ya sea desde una base de datos externa o de forma directa conectando los diodos al torso del paciente, para mediante técnicas de Machine Learning detectar las anomalías en la señal. Este sistema está conformado por: un sensor ECG, un conversor A/D y un micro procesador

La etapa de procesamiento de información se realizó en una tarjeta raspberry pi (Micro procesador) y usando como interprete de programación Python, la señal de entrada se filtra mediante FOURIER, WAVELET y BUTTHEWORT. Para la clasificación de ondas y detección de anomalías se utilizó el método K-NN(K Nearest Neighbours), a continuación se muestran los resultados obtenidos.

Adquisición de datos provenientes de los contadores T1-T2-T3-T4

-- Bella Rosa --- 2019 --

Debido a la necesidad de mantener un registro permanente del número de mallas que ingresan a la etapa de poscosecha provenientes de las líneas de transporte áereo (T1-T2-T3-T4), se precento una alternativa para mejorar el sistema de conteo el cual estaba compuesto por contadores digitales de 6 dígitos instalados en las afueras del area de poscosecha, estos contadores presentaban inconvenientes como: el registro y reseteo se realizaban de forma manual, además su precisión oscilaba en un 85-90%.

Por estos motivos se planteó llevar la información entregada por los contadores a un entorno virtual, en el cual se mejor su precisión a un 97% por medio de incorporación de filtros digitales y de la linealización del sensor, además su registro y reseteo se realiza de forman automática por medio de una aplicación instalada en la computadora principal del área de poscosecha.

Automatización del proceso de fumigación

-- Bella Rosa y Rose Connection -- 2019/2020 --

Una etapa crítica dentro de la producción de flores es el proceso de fumigación de mallas al ingreso del área de poscosecha, pues esta consume una alta cantidad de producto fumigante representando un gasto considerable, para minimizar estos gastos se planteó el desarrollo de un sistema de fumigación automático por el cual se suministra una cantidad justa de producto.

Este sistema consta de un segmento móvil el cual se desplaza longitudinalmente con ayuda de un motorreductor inyectando el producto a las mallas mediante la activación y desactivación de electroválvulas. Todo este proceso se encuentra controlado por medio de un logo de la familia SIEMNS. Con ayuda de este sistema se espera una reducción hasta un 37% de consumo de producto.

Sistema automático de corte de tallos

-- Bella Rosa y Rose Connection -- 2020 --

Dentro del proceso de tinturado es necesario realizar cortes en los tallos que permitan la absorción de diferentes colorantes, este proceso se realiza de forma manual consumiendo muchos recursos humanos para optimizar este proceso se planteó la construcción de una maquina automática que realice el corte de los tallos

La máquina está compuesta por una banda transportadora sobre la cual se colocan las rosas para posteriormente ingresar en a la sección de corte que mediante dos cilindros neumáticos ajustan el tallo y realizan el corte. Esta máquina es controlada por un PLC DELTA.

Plan de mantenimiento industria 4.0

-- Metálicas FORZA -- 2018--

El área de mantenimiento dentro de una empresa es fundamental y se precisa de la incorporación de mejoras continuas que ayuden a disminuir el tiempo de paro de un equipo. Es por ello que se diseño un plan de mantenimiento predictivo enfocado a la INDUSTRIA 4.0 (el internet de las cosas) por medio de la instalación de sensores con tecnología IO-Link se recopila información en tiempo real que ayuda a realizar un diagnóstico efectivo del estado del equipo, identificando así anomalías en su funcionamiento.

Para la adquisición de la información de los sensores se utilizó una tarjeta raspberry pi la cual agrupa la información y genera reportes de forma continua anexándolos a una base de datos en la nube para facilitar el acceso de esta información.

Control para parqueadero de bicicletas autosustentable

-- Universidad Técnica del Norte -- 2018 --

Para la FACULTAD DE INGENIERIA EN CIENCIAS APLICADAS FICA de la UNIVERSIDAD TÉCNICA DEL NORTE se diseñó un parqueadero de bicicletas autosustentable implementando paneles solares y mecanismos de seguridad eléctricos controlados por una tarjeta ARDUINO MEGA. El accionamiento de los mecanismos de seguridad está dado por un passsword introducido por el usuario.
